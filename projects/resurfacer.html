Resurfacer

Web Productivity HTML

You save links with the best intentions. Something interesting appears, you think "I'll read this later," and it joins the pile. Resurfacer is built on the honest acknowledgment that "later" is a real time and you actually need those links back when it arrives.

Save a link with a note about why it matters and when you'll want it. That context is what separates a link graveyard from something useful. It resurfaces them instead of letting them disappear into a folder you never open.
